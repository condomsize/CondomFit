Modèle physique utilisé
Modèle néo-hookéen · C₁ = E/6 (G = E/3, ν = 0,5)
Membrane libre axialement · σ_z = 0 → λ_z = λ_θ−½ · λ_r = λ_θ−½
Vérification incompressibilité : λ_r·λ_θ·λ_z = λ−½·λ·λ−½ = 1 ✓
σ_θ = 2C₁(λ_θ² − λ_θ−1) · t = t₀/√λ_θ · T = 2C₁t₀(λ_θ3/2 − λ_θ−3/2)
ΔP = T / R avec R = C_pénis / (2π) [Loi de Laplace — cylindre]
