Modèle physique complet : pression d'un préservatif sur un pénis en érection — cas de la membrane libre axialement (σ_z = 0).

1 · Hypothèses du modèle

  • Géométrie : le pénis est modélisé comme un cylindre rigide de rayon R = C/(2π), où C est la circonférence érectile.
  • Membrane mince : le préservatif est une membrane cylindrique mince d'épaisseur initiale t₀ ≪ R. La loi de Laplace s'applique.
  • Matériau : latex naturel, modélisé comme un solide néo-hookéen incompressible. Densité d'énergie : W = C₁(I₁ − 3), avec C₁ = E/6 (relation exacte pour ν = 0,5).
  • Déformation homogène : les élongations sont uniformes sur toute la surface.
  • Condition axiale : le préservatif est libre de se déplacer axialement — aucune force axiale n'est imposée, donc σ_z = 0.
  • Condition radiale : la surface externe du préservatif est libre (pas de contact externe), donc σ_r = 0.

2 · Démonstration pas-à-pas

1

Élongations principales

Le préservatif (circonférence initiale C₀ = 2w) est étiré sur le pénis (circonférence C). On définit :

λ_θ = C / C₀ = C / (2w)  [élongation circonférentielle, > 1]
λ_z = L / L₀  [élongation axiale, à déterminer]
λ_r = t / t₀  [élongation radiale = amincissement]

Incompressibilité : λ_r · λ_θ · λ_z = 1, donc :

λ_r = 1 / (λ_θ · λ_z)
2

Modèle néo-hookéen — tenseur de Cauchy

Pour un matériau néo-hookéen incompressible, la contrainte de Cauchy dans la direction principale i est :

σ_i = 2C₁ λ_i² − p

où p est un multiplicateur de Lagrange (pression hydrostatique). Explicitement :

σ_r = 2C₁ λ_r² − p = 2C₁ / (λ_θ² λ_z²) − p
σ_θ = 2C₁ λ_θ² − p
σ_z = 2C₁ λ_z² − p
3

Condition radiale : σ_r = 0

La surface externe est libre, donc σ_r = 0 :

0 = 2C₁ / (λ_θ² λ_z²) − p ⟹ p = 2C₁ / (λ_θ² λ_z²)
4

Condition axiale : σ_z = 0

Le préservatif est libre axialement, donc σ_z = 0 :

0 = 2C₁ λ_z² − p = 2C₁ λ_z² − 2C₁ / (λ_θ² λ_z²)

En divisant par 2C₁ :

λ_z² = 1 / (λ_θ² λ_z²) ⟹ λ_z⁴ = 1/λ_θ² ⟹ λ_z² = 1/λ_θ
λ_z = λ_θ−½ = 1/√λ_θ
5

Élongation radiale λ_r et épaisseur courante

Par incompressibilité :

λ_r = 1 / (λ_θ · λ_z) = 1 / (λ_θ · λ_θ−½) = λ_θ−½

Vérification incompressibilité :

λ_r · λ_θ · λ_z = λ_θ−½ · λ_θ · λ_θ−½ = λ_θ0 = 1 ✓

L'épaisseur courante est donc :

t = t₀ · λ_r = t₀ / √λ_θ
6

Contrainte circonférentielle σ_θ

En substituant p et λ_z = λ_θ−½ :

σ_θ = 2C₁ λ_θ² − 2C₁ / (λ_θ² · λ_z²) = 2C₁ λ_θ² − 2C₁ / (λ_θ² · λ_θ−1)
σ_θ = 2C₁ (λ_θ² − 1/λ_θ) = 2C₁ (λ_θ² − λ_θ−1)
7

Tension membranaire T

T = σ_θ · t = 2C₁ (λ_θ² − λ_θ−1) · t₀/√λ_θ
T = 2C₁ t₀ (λ_θ3/2 − λ_θ−3/2)
8

Pression via la loi de Laplace — cylindre

Pour une membrane cylindrique mince, la loi de Laplace donne ΔP = T/R, avec R = C/(2π) :

ΔP = T / R = [2C₁ t₀ / R] · (λ_θ3/2 − λ_θ−3/2)

avec C₁ = E/6, λ_θ = C/(2w), R = C/(2π)

5 · Vérification numérique — mesure de Gerofi

Données expérimentales — Gerofi et al.

J. Gerofi, F. Deniaud, P. Friel — Interaction of Condom Design and User Techniques and Condom Acceptability

Gerofi a mesuré la pression exercée par deux préservatifs sur un mandrin cylindrique rigide (C = 135 mm) :

Préservatif WHO-Hankook (modèle Hankook, fourni par le WHO)

  • Largeur nominale : w = 51,5 mm → C₀ = 103,0 mm
  • Élongation : λ_θ = 135/103,0 = 1,311
  • Épaisseur : t₀ = 0,069 mm · R = 21,49 mm
  • Pression mesurée : ΔP = 2,3 kPa
ΔP = [2C₁ t₀ / R] · (λ3/2 − λ−3/2)
= [2C₁ × 6,9×10−5 / 0,02150] × (1.3113/2 − 1.311−3/2)
= C₁ × 5.357×10−3 × (1.501 − 0.666)
= C₁ × 5.3573×10−3
C₁ = 2300 / 5357 = 0.4293 MPa  ⟹  E = 2.58 MPa

Préservatif USAID-Sultan (modèle Sultan, fourni par le USAID)

  • Largeur nominale : w = 52,8 mm → C₀ = 105,6 mm
  • Élongation : λ_θ = 135/105,6 = 1.278
  • Épaisseur : t₀ = 0,061 mm · R = 21,49 mm
  • Pression mesurée : ΔP = 2,0 kPa
ΔP = [2C₁ t₀ / R] · (λ3/2 − λ−3/2)
= [2C₁ × 6,1×10−5 / 0,02150] × (1.2783/2 − 1.278−3/2)
= C₁ × 4.2792×10−3 × (1.445 − 0.692)
= C₁ × 4.2792×10−3
C₁ = 2000 / 4279 = 0.4674 MPa  ⟹  E = 2.80 MPa

Les deux back-calculations donnent E ∈ [2,6 ; 2,8] MPa, cohérent avec la littérature pour le latex de préservatif (1,5 – 4 MPa), validant le modèle.
